# Invest Analyzer

股票分析 Skill，提供实时行情、技术指标分析和新闻资讯获取功能。

## 功能特性

- **实时行情**: 获取股票当前价格、涨跌幅、成交量等基础数据
- **K线数据**: 支持多种时间周期（5分钟、15分钟、30分钟、1小时、日线）
- **技术指标**: 计算 MA、MACD、KDJ、RSI、BOLL 等常用技术指标
- **新闻资讯**: 多源新闻抓取，智能关键词扩展
- **技术评级**: 基于后端技术面规则给出买入/增持/持有/减持/卖出判断
- **一键快照**: 单次调用返回行情、日线、5分钟、指标和技术评级
- **综合判断**: 将技术评级与新闻影响分开呈现，避免混淆

## 快速开始

### 安装

以 Claude Code 为例，首先进入 skills 目录：

```bash
cd ~/.claude/skills
git clone https://github.com/zhangyangx/invest-analyzer.git
```

### 使用方式

在支持 AI 编程助手的工具中（如 Claude Code），直接对话即可：

```
分析一下 600519
```

```
分析一下 贵州茅台
```

```
帮我看看 600519 最近有什么消息
```

```
贵州茅台（600519）结合最新消息分析一下
```

### 支持的对话方式

| 用户输入 | AI 行为 |
|---------|---------|
| `分析 600519` | 行情 + 技术指标分析 |
| `分析 贵州茅台` | 先按关键词搜索股票代码，再做行情 + 技术指标分析 |
| `600519 怎么样` | 行情 + 技术指标分析 |
| `看看 600519 的新闻` | 行情 + 技术指标 + 新闻分析 |
| `600519 最近有消息吗` | 行情 + 技术指标 + 新闻分析 |
| `600519 财报相关消息` | 行情 + 技术指标 + 财报主题新闻 |

### 股票名称/代码格式

- 支持股票名称或关键词搜索（如 `贵州茅台`、`茅台`）
- 搜索到多个结果时，需要确认具体股票代码

### 股票代码格式

- `600519` - 自动识别市场（上海）
- `sh600519` - 指定上海市场
- `sz000001` - 指定深圳市场
- `000001` - 自动识别市场（深圳）
